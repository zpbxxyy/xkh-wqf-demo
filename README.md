# 侠客汇 × 微企付 · 3C数码支付Demo

> 双角色交互演示：店员PC端选品下单 + 客户手机端扫码支付

## 演示地址

https://zpbxxyy.github.io/xkh-wqf-demo

## 场景说明

**侠客汇** 是一个3C数码交易平台，本Demo演示了店员使用微企付为客户完成支付的完整流程。

### 角色一：店员端（PC）
- 选择商品类型（笔记本电脑/手机/平板）
- 选择成色（全新~七成新）
- 填写品牌型号、价格、客户信息
- 提交订单，生成微企付H5支付二维码

### 角色二：客户端（手机）
- 微信扫描二维码
- 进入微企付收银台，确认订单信息
- 选择银行卡，跳转银行APP
- 输入密码完成支付
- 返回支付成功结果

## 技术特点

- 纯前端单HTML文件，零依赖
- Canvas伪二维码生成
- 多屏幕切换模拟完整支付链路
- 自动密码填充动效
- 响应式布局，支持移动端访问
